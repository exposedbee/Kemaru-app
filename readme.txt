To launch the game run the main.py file

Game features:

There are three game modes: easy, medium and hard

	Easy: 6*6 grid
	      Hint available

	Medium: 9*9 grid
		Hint available

	Hard: 9*9 grid
		Hint not available

Once game starts: timer starts running and once you complete the game successifully,
			winner page is displayed with three options: view your solution, play again and quit

Note:
	Hint can be activated and de-activated by clicking on it(once hint activated it turns red)
	Hint is only displayed when you have filled a inner grid entirelt. (Hint shows which all entries are incorrect in
	that perticular inner grid)
